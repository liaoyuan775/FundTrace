# FundTrace 验收清单

- [ ] 案件、受害人和案件目录按 `case_id` 隔离。
- [ ] 上传文件保存 SHA-256，重复材料明确提示。
- [ ] ZIP 路径穿越、数量和解压体积受到限制；RAR 缺少工具时明确失败。
- [ ] CSV、XLSX、DOCX、PDF和图片保留来源页、工作表或行号。
- [ ] 模型输出只进入待校核状态，不直接生成正式结果。
- [ ] 未确认草稿不能生成 CSV 版本或设置涉诈起点。
- [ ] FIFO、保守、可能上限、比例分摊均不超过受害人起点金额。
- [ ] 拓扑支持缩放、平移、拖动、布局切换和扩散播放。
- [ ] 节点、边、流水和证据详情联动。
- [ ] 原始流水常驻，支持全文查询、金额筛选和 CSV 导出。
- [ ] 任一分析边可追踪到原始交易 ID 和材料来源。
- [ ] 10 万笔流水图聚合在基准阈值内完成。
