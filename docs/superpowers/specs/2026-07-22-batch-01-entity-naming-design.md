# Batch 01 实体命名重建设计

## 目标

重建 `test-data/batch-01/materials` 内全部资金主体名称，消除“受害人甲”“张某一”“商户A”等占位式命名，使十份材料在演示中更接近真实调证材料。所有人物、企业、商户和出口主体均为虚构；真实银行及支付机构名称保留。

本次只改变名称和相应材料标题，不改变120笔唯一交易、15笔跨文件重复、账号、流水号、时间、金额、余额、渠道、证据分布和拓扑结构。

## 生成与同步

实体名册是唯一名称来源，写入合成材料生成脚本。生成器据此重新生成两份 CSV、两份 Excel、两份 PDF、两份 Word 和两张图片，并同步更新 `ground_truth.json`、`distribution.json`、`file_manifest.csv` 和测试报告中的名称与文件引用。

不得直接在生成物中做孤立替换。每个账号在所有材料和真值中必须始终映射到同一名称；15个重复组中的同一流水也必须保持相同双方名称。

## 验收

- 十份材料及真值中不再出现“受害人甲/乙/丙/丁”“某一至某十六”“商户A/B”等旧占位名称。
- 32个账号与新实体名册一一对应，没有同账号异名或同一交易跨文件名称冲突。
- 仍为120笔唯一交易、135条材料记录和15个重复组。
- 拓扑仍为32个节点，交易总额、边数及所有账号间关系不变。
- 通过上传接口新建隔离案件并重新建模；结构化材料、文档和图片分别报告结果，不将视觉接口失败伪装为材料成功。
